Exercício 35 - elemento de texto com bloco de código

...Mas o que você pode estar se perguntando após todos esses estudos é:

Quais são as diferenças entre os modos de escrita das linguagens de programação?

Pois bem, vou te mostrar com um exemplo bem simples, o Hello World!

    
        Hello World em Java

        public class HelloWorld {
            public static void main(String[] args) {
                System.out.println("Hello World!");
            }
        }
    

    
        Hello World em Phyton

        print("Hello World!")
    

    
        Hello World em JavaScript

        console.log("Hello World!");
